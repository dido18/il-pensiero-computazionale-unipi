Python Blocks:

upper
comando: str.upper()
trasforma una stringa in magliuscolo
Esempio:

str = "ciao"
str.upper()
print(str) #output CIAO

isupper
comando: str.isupper()
verifica se una stringa è tutta in magliuscolo
Esempio:

if str.isupper():
    # la stringa è tutta magliuscola
else:
    # non tutta la stringa è magliuscola

ord
comando: ord('A')
restituisce il codice ASCII associato a una lattera
Esempio:

valore = ord('A')
print(valore) #output 65

chr
comando: chr(65)
restituisce il carattere associato al codice ASCII
Esempio:

valore = chr(65)
print(valore) #output A
